<h1><center>数据库结构</center></h1>

* ## 用户表(users)
    - ### user_id
    - ### wx_code    微信
    - ### pwd        密码
    - ### study_code 学号
    - ### scholl_id  所在学校
    - ### class      所在班级
    - ### desc       描述
    - ### image      头像
    - ### type       类型(1: 学生)
    - ### value      学分
    - ### integral   积分
    - ### name       姓名

* ## 任务表(task)
    - ### task_id  
    - ### scholl_id 学校id  
    - ### tag       类型  
    - ### content   内容  
    - ### starttime 开始时间  
    - ### endtime   结束时间
    - ### value     学分
    - ### pass      及格线
    - ### receiver  接收者

* ## 个人运动信息(peopleinfo)
    - ### peopleinfo_id
    - ### task_id   任务id
    - ### user_id   用户id
    - ### fraction  分数
    - ### starttime 开始时间
    - ### endtime   结束时间
    
* ##  实时信息(news)
    - ### news_id
    - ### top       权重
    - ### user_id   用户id
    - ### content   内容
    - ### tag       标签
    - ### time      时间

* ##  评论(comment)
    - ### comment_id
    - ### user_id   用户id
    - ### time      时间
    - ### content   内容
    - ### news_id   信息Id

* ##  学校(scholl)
    - ### scholl_id
    - ### name      学校名称
    - ### adminname 管理员账号
    - ### adminpwd  管理员密码
